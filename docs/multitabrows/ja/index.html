📖 概要

Multi Tab Rows（MultiTabRows@Merci.chao.uc.js）は、Firefox における多段タブの完全な実装です。userChrome.js スクリプトとして開発され、クラシックな MultiRowTabLiteforFx.uc.js（Izheil 氏による改修版）や多機能な Tab Mix Plus に着想を得て、洗練された操作性と快適さを備え、一番の多段タブを目指しています。

🧩 互換性

  • Firefox 115 および最新リリース版に対応
  • Windows 7 から Windows 11 までサポート

🌟 注目ポイント

📐 スペース活用の最適化

Multi Tab Rows はスペースを余すことなく活用し、ウィンドウ操作ボタン下の領域まで広げます。

クラシックな見た目を好む方には、タブバーをナビゲーションバーの下に配置することも可能です。

🖱️ ドラッグ＆ドロップ操作

独自のドラッグ＆ドロップ機能で、タブ管理は直感的かつスムーズ。インジケーター不要で、多段モードでも自然に動作します。

🚀 最新機能のサポート

タブグループや分割ビューを利用していても、Multi Tab Rows は Firefox の最新機能に完全対応。タブや分割ビューをドラッグしてグループ化する操作も、従来通り直感的で自然に動作します。もしグループ化があまり好みでない場合は、設定リストにある「タブを他のタブにドロップした際にグループ化」機能は、オフにすることもできます。

✨ 洗練された操作感

Firefox 本来のタブ挙動を洗練し、不具合を解消して不足を補うことで、よりスムーズで完成度の高い体験を実現します。すべての操作は、まるで Firefox に標準搭載されているかのように自然です。

📌 ピン留めタブのグリッド表示

ピン留めされたタブはコンパクトに整理され、グリッド状に並ぶため、タブバーがスクロール可能になってもすぐにアクセスできます。

通常のタブのようにスクロールで流れる挙動を好む方は、この機能をオフにしてスペースを節約することも可能です。

🛠️ お好きなように調整

デフォルト設定はすでに最適化されていて、すぐに使えます。さらに Multi Tab Rows は、タブサイズからレイアウト、操作まで幅広い設定を備え、好みに合わせて自由に調整できます。

🔄 自動更新チェック

新しい Firefox に切り替えても、スクリプトが突然壊れる心配や更新を探し回る煩わしさはありません。Multi Tab Rows は自動更新確認機能を備えており、ワンクリックで最新の改善や互換性をすぐにゲットできます。より細かくコントロールしたい方には、手動更新や無効化も選べます。

🛡️ 安心のサポート

Multi Tab Rows は熱意と長期的な目標を持った取り組みから生まれました。長年の拡張機能開発で培った経験を背景に、これからも安心のサポートを続けてまいります。

👤 開発者について

2007 年から、私は数々の Firefox 拡張機能を開発してきました：

  • Personal Menu
  • Movable Firefox Button
  • Pick & Save Images
  • History Submenus II
  • Page Title in URL Bar
  • Semi-Full Screen
  • Float Toolbars in Full Screen
  • Mid-Click Restart

最初の Personal Menu は、Firefox のカスタマイズを愛するユーザー向けのニッチな拡張機能でしたが、ピーク時に 16 万人以上のユーザーに利用されました。Personal Menu は 9 年間にわたりメンテナンスを続け、さらに他の 2 つの拡張機能も AMO（addons.mozilla.org）でおすすめとして紹介されました。これらはいずれも UI 調整を目的としたツールで、唯一 Pick & Save Images だけは画像取得に特化し、Firefox の UI に自然に溶け込む多機能なボタンを備えていました。

これらの拡張機能は、WebExtensions がブラウザー UI のカスタマイズをサポートしていないため、Quantum 時代（Firefox 57+）では生き残ることができませんでしたが、一部の機能は userChrome.js を通じて今も生き続けています。興味のある方はぜひ私の GitHub をご覧ください。

そして現在は、数千時間を費やして取り組んできたプロジェクト Multi Tab Rows に注力して、できる限り続けていきます。
