📖 概要

Multi Tab Rows（MultiTabRows@Merci.chao.uc.js）は、Firefox における多段タブの徹底的かつ完全な実装です。userChrome.js スクリプトとして開発され、クラシックな MultiRowTabLiteforFx.uc.js（Izheil 氏版と多段タブ.zip）や多機能な Tab Mix Plus に着想を得て、洗練された操作性と快適さを備え、全新・究極の多段タブを差し上げます。

🧩 互換性

  • Firefox 115 および最新リリース版をサポート
  • Windows 7～11 および Ubuntu をサポート

🌟 注目ポイント

📐 スペース活用を最大化

Multi Tab Rows はスペースを余すことなく活用し、ウィンドウ操作ボタン下の領域まで広げます。

クラシックな見た目を好む方には、タブバーをナビゲーションバーの下に配置することも可能です。

タブがスペースを取りすぎて圧迫感まで感じる体なので、一番コンパクトなのを頼む？大丈夫だ、問題ない！ただし、息つく余裕を忘れずに、コンパクトさと快適さのちょうどいいバランスに整えてくださいよ。

🖱️ ドラッグ＆ドロップ操作

独自のドラッグ＆ドロップ機能で、タブ管理は直感的かつスムーズ。インジケーター不要で、多段モードでも自然に動作します。

🚀 最新機能のサポート

タブグループや分割ビューを利用していても、Multi Tab Rows は Firefox の最新機能に完全対応。タブや分割ビューをドラッグしてグループ化する操作も、従来通り直感的で自然に動作します。

もしグループ化があまり好みでない場合は、設定リストにある「タブを他のタブにドロップした際にグループ化」機能は、オフにすることもできます。

✨ 洗練された操作感

Firefox 本来のタブ挙動を洗練し、さまざまな細やかな不具合を解消して不足を補うことで、よりスムーズで完成度の高い体験を実現します。すべての操作は、まるで Firefox に標準搭載されているかのように自然です。

📌 ピン留めタブのグリッド表示

ピン留めされたタブはコンパクトに整理され、グリッド状に並ぶため、タブバーがスクロール可能になってもすぐにアクセスできます。

通常のタブのようにスクロールで流れる挙動を好む方は、この機能をオフにしてスペースを節約することも可能です。

🛠️ お好きなように調整

デフォルト設定はすでに最適化されていて、すぐに使えます。さらに Multi Tab Rows は、タブサイズからレイアウト、操作まで幅広い設定を備え、好みに合わせて自由に調整できます。

🔄 自動更新チェック

新しい Firefox に切り替えても、スクリプトが突然壊れる心配や更新を探し回る煩わしさはありません。Multi Tab Rows は自動更新確認機能を備えており、ワンクリックで最新の改善や互換性をすぐにゲットできます。

より細かくコントロールしたい方には、手動更新や無効化も選べます。

🛡️ 安心のサポート

Multi Tab Rows は熱意と長期的な目標を持った取り組みから生まれました。長年の拡張機能開発で培った経験を背景に、これからも安心のサポートを続けてまいります。

👤 開発者について

2006 年から、私は数々の Firefox 拡張機能を開発してきました：

  • Personal Menu
  • Pick & Save Images
  • Movable Firefox Button
  • History Submenus II
  • Page Title in URL Bar
  • Semi-Full Screen
  • Float Toolbars in Full Screen
  • Mid-Click Restart

（上記リンクは web.archive.org のアーカイブです）

最初の Personal Menu は、Firefox のカスタマイズを愛するユーザー向けのニッチな拡張機能でしたが、ピーク時には 165,000 人以上 のユーザーに利用され、世界中のボランティアによって 30 の言語に翻訳されました。ツールバーにボタンを備え、そのメニューを自由にカスタマイズできるようにし、よく使う項目を配置して従来のメニューバーを完全に隠すことができました。この拡張機能は Firefox 1.5 でデビュし、メニューバーをオレンジ色のボタンに置き換えた Firefox 4 よりも、なんと 5 年も先駆けていました！（ちなみに、その頃 IE7 はまだベータ版で、Chrome が出てくるのはさらに 2 年後でした。）

Personal Menu は 9 年間積極的にメンテナンスされ、その後も安定性を維持しました。さらに、他の 2 つの拡張機能も、公式の Firefox アドオンサイトでおすすめとして紹介されました。

これらはすべて UI 調整用のツールでしたが、Pick & Save Images だけは違いました。ほかの拡張機能とは異なり、画像の取得に特化し、Firefox Sync に対応したプロファイルベースの設定を提供し、初心者向けにもプロ向けにも対応したフィルタリング機能を備えていました。さらに、Firefox の UI に自然に溶け込む、洗練されて多用途に使えるボタンも備えていたのです。また、当時 Firefox 用 3 つの人気画像保存拡張機能（有名な DownThemAll! を含む）に戦いを挑むための比較表（英語）を作成しました。闘技場として、ユーザー向けの機能から深い技術的なサポートや処理まで、24 の客観的な項目で勝負しました。

残念ながら、これらの拡張機能は Quantum 時代（Firefox 57 以降）を越えて存続できませんでした。WebExtension ではブラウザーの UI 改修が認められていないためです。それでも、小さな UI 調整用の拡張機能のいくつかは、userChrome.js を通じて今も生き続けています。興味のある方はぜひ私の GitHub をご覧ください。

そして現在は、数千時間を費やして取り組んできたプロジェクト Multi Tab Rows に注力して、できる限り続けていきます。
